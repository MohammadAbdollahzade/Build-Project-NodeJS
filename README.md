> <b>1.  **Create project folder**</b>
```shell
mkdir backend
```

> <b>2.  **Go to folder**</b>
```shell
cd backend
```

> <b>3.  **Open folder in vscode**</b>
```shell
code .
```

> <b>4.  **The command to create the package.js file**</b>
```shell
npm init -y
```
